CSS Animations

what are css animations?

An animation lets an element gradully change from one style to another. You can change many css propities you want, as many times as you want. to use css animation you must first specify some keyframes for the animation.Key frames hold what styles the element will have at certin times.

the @keyframes rule:

1.when you specify css styles inside the @keyframes rule, the animation will gradually change from the current style to the new style at certain

2.To get an animation to work you must bind the animation to an element

specify the speed curve of the animation

The animation-timing-funtion propity specifies the speed curve of the animation. The animation-timing-funtion propity can have the following values

1.ease-Specifies an animation with slow start then fast then end slowly(this is defalt)

2.linear-Specifies an animation with the same speed from start to end

3.ease-in-Specifies an animation with with the same speed from start to end

4.ease-out-Specifies an animation with a slow start

5.ease-in-out-Specifies an animation with a slow start and end
